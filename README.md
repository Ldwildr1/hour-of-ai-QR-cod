# hoai-qrcode
This repository is a static webpage for generating a QR Code from a URL of one line text.
There is a box for inputting the text and the generated QR code is shown bellow. A button,
underneath, lets you download the code as a PNG image. This code us generated by an AI agent,
as an example of how to use it. 

Testing
=======

Serve this folder with a simple HTTP server and open the page for testing. 
This repostiory is deployed as a static webpage using the default Github pages
workflow.
